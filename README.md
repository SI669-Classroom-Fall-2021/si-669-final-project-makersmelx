# Setup
Create a file named `secret.ts` at `/src`. and fill in with the firebase auth key.
~~~typescript
export default {
  apiKey: '...',
  authDomain: '...',
  projectId: '...',
  storageBucket: '...',
  messagingSenderId: '...',
  appId: '...',
  measurementId: '...',
};
~~~
